go-sqlite is a SQLite3 Toolbox/Package for GNU Octave and Matlab!

KEYFEATURES

  • supports full automatic save and load:
    • n-dimensional matrix ✔
    • 2D nested cells ✔
    • struct arrays ✔
    • chars ✔
  • automatic variablename as table name (optional: name your tablename)
