# Index

__go-sqlite__ is a SQLite3 Toolbox/Package for GNU Octave __and__ Matlab!  

# KEYFEATURES

* supports full automatic save and load:
	* n-dimensional matrix ✔
	* 2D nested cells ✔
	* struct arrays ✔
	* chars ✔
* automatic variablename as table name _(optional: name your tablename)_
